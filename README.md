# 01_brief

Vibe coding quickstart workspace.

## Environment check

- Node.js: `node --version`
- npm: `npm --version`
- pnpm: `pnpm --version`
- yarn: `yarn --version`
- Python: `python --version`
- pip: `pip --version`
- uv: `uv --version`

## Start options

### Node project

```bash
npm init -y
```

or

```bash
pnpm init
```

### Python project

```bash
uv venv
```

Activate on PowerShell:

```powershell
.\.venv\Scripts\Activate.ps1
```
